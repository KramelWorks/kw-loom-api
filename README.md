# 🚀 KW Loom API

Kramelworks' simple Content Management System (CMS) API, designed for internal and educational use.

---

## 📌 Overview

This project provides a structured backend API built with a focus on:

- Clean architecture
- Separation of concerns
- Scalability and maintainability
- Consistent patterns across the codebase

---

## 🏗️ Architecture

The project follows a layered architecture:

- **Domain** → business rules
- **Application** → use cases and orchestration
- **Infrastructure** → HTTP, database, external services

---

## ⚙️ Getting Started

### 📦 Installation

```bash
npm install
```

### ▶️ Run the project

```bash
npm run dev
```

By default, the server runs on:

```
http://localhost:5001
```
